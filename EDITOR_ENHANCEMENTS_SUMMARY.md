## 💡 Usage Tips

1. **Select text first** before applying colors or creating links
2. **Click the ? button** for quick reference and pro tips
3. **Use Ctrl+B, Ctrl+I** for quick bold and italic
4. **Images can be uploaded** or added via URL
5. **Tables support** resizing and header rows
6. **Code blocks** automatically highlight syntax

## 🔧 How to Use

### In the Blog Creation Form

1. **Navigate to:** `/dashboard/blogs/create`
2. **Look for:** "Content" field with enhanced toolbar
3. **Available actions:**
   - Click toolbar buttons to apply formatting
   - Select text, then apply colors or create links
   - Upload images or insert from URL
   - Create tables for data
   - Add code blocks for technical content
4. **Press the ? button** for help anytime

### File Uploads via Form

- **Cover Image:** Upload featured image (5MB max)
- **Gallery Images:** Add up to 6 images (5MB each)
- **Attachments:** Add files up to 50MB each
- **Video/Audio:** Upload or add URLs
